# Module 3 - Java

This module contains Java programming exercises and source code.

## Structure
- `com/` - Java package directory containing application source files and exercise code.

## How to use
1. Open the Java source file you want to run.
2. Compile using a Java compiler, for example:
   ```bash
   javac -d out src/YourMainClass.java
   ```
3. Run with:
   ```bash
   java -cp out com.example.YourMainClass
   ```

## Notes
- The `com` directory likely contains package-based Java classes for exercises.
- Use a modern JDK compatible with the source code version.
