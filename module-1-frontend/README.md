# Module 1 - Frontend

This module contains frontend web development exercises for HTML5, CSS3, Bootstrap, and JavaScript.

## Structure
- `Assets/` - Shared images and videos used by frontend exercises.
- `BOOTSTRAP/` - Bootstrap 5 exercises covering layout, components, utilities, icons, and JavaScript plugins.
- `CSS3/` - CSS3 practice exercises for selectors, layouts, animations, and responsive styling.
- `HTML5/` - HTML5 exercises for semantic markup, forms, media, and page structure.
- `JAVA SCRIPT/` - JavaScript exercises for DOM manipulation, events, functions, and browser scripting.

## How to use
1. Open the folder for the exercise you want to view.
2. Open the HTML file in a browser.
3. Use the browser DevTools to inspect layout and behavior.

## Notes
- Most exercises are static and can be run locally without a web server.
- If an exercise includes Sass, compile it using a Sass tool before opening the HTML file.
